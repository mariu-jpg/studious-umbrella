⚡ スプレッドシート連携の設定
1
Google スプレッドシートを新規作成 → メニュー「拡張機能」→「Apps Script」を開く
2
下のコードを全選択して貼り付け → 保存（Ctrl+S） function doPost(e) { const ss = SpreadsheetApp.getActiveSpreadsheet(); const d = JSON.parse(e.postData.contents); const sheetName = d.sheet || '家計簿'; let sheet = ss.getSheetByName(sheetName); if (!sheet) { sheet = ss.insertSheet(sheetName); if (sheetName === '家計簿') { sheet.appendRow(['日付','曜日','記入者','カテゴリ','決済','金額','メモ']); } else { sheet.appendRow(['日付','曜日','記入者','種別','金額','メモ']); } sheet.setFrozenRows(1); } if (sheetName === '家計簿') { sheet.appendRow([d.date,d.weekday,d.user,d.category,d.payment,d.amount,d.memo]); } else { sheet.appendRow([d.date,d.weekday,d.user,d.savType,d.amount,d.memo]); } return ContentService .createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); }
3
「デプロイ」→「新しいデプロイ」→ 種類：ウェブアプリ → アクセス権：全員 → デプロイ → URLをコピー
4
URLをここに貼り付けて「保存」
